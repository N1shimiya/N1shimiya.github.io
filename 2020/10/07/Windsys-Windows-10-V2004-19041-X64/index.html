系统介绍

  • 移除部分预装UWP应用（如需使用可到应用商店重新安装）
  • 无精简任何系统组件
  • 云上自动构建，随着 Windows 更新而更新
  • 优化系统设置，改善使用体验
  • 不关闭 UAC
  • 修改系统自带的记事本为 NotePad2-mod
  • 驱动总裁 1.9.21.0（系统部署时自动安装硬件驱动）(可选)
  • UserDirChanger（智能调整桌面，我的文档，图片等目录到D盘（可选））
  • 移除 Onedrive（如有需要可到系统内 Windsys 工具箱重新安装）
  • 启用统一写入过滤器功能（巨硬官方功能，类似于冰点还原，可通过工具箱 UWF 管理器 调用）
  • 内置浏览器默认主页均为其默认启动页或空白页!
  • Windows Update 保持默认的自动更新 (如需关闭工具箱有相应工具)
  • 保留 Windows Defender（默认为禁用，可使用工具箱软件启用）
  • WTDR.AntiFamily （尽可能防止流氓PE或安装器预装垃圾软件）
  • 安装时可选 自建用户(可登陆 Microsoft 账号) 或 默认 Administrator 用户
  • WTDR 系统部署辅助工具
  • 支持通过 WTDR.Pack“预装” Office
  • 可自定义需预装的软件
  • 还原原系统盘符功能（需要提前生成记录文件，可看下面相关教程）
  • 支持使用微软官方方式从旧版系统升级到该系统（强烈不建议）
  • ……

其他说明

  • 该第三方系统无病毒无推广无流氓，如出现问题请自查，我不背锅
  • ↑已知有几个事业单位、学校、企业、等有在用，咱哪敢搞事情呀
  • 推荐使用 Edgeless、FirPE、Hikari PE、微PE 等优秀 PE 进行系统安装
  • 如被国内杀软提示有病毒，那大概都是误报，同时建议使用国外优质杀软
  • 该版本不提供Wim格式镜像，如需进行二次挂载修改请自行转换，ESD格式跟Wim格式的安装方式是一样的
  • 如需加入“预览体验计划”请到工具箱打开“修复预览体验计划和强迫症设置”来启用一下遥测服务
  • 如当前系统使用正常，无需下载下来重装更新，只需要下载新系统镜像(Wim,Esd)下来替换(放在U盘里的)旧系统镜像(Wim,Esd)就行
  • 驱动总裁版请在硬盘空余空间有15G左右大小下安装！
  • 该镜像会持续进行更新，推荐定时重新下载并替换旧系统镜像
  • 不推荐任何新手或无动手能力的使用，否则你会遇到重重无法解决的问题！
  • 下载前请务必阅读下载页面中的“使用须知”！

镜像说明

不带驱动版为不带 驱动总裁 的版本，系统本身内置的驱动还是有滴

镜像文件名尾部不带 DrvCeo 的为不带驱动版本

镜像文件名尾部带 DrvCeo 的为带驱动版本

安装时请注意分卷选择：

带 Administrator 的分卷为 默认使用 Administrator 账户登录

带 New Account 的分卷为 系统部署 时自定义账户或使用 MicroSoft 账户登录

带 DrvCeo 的为 驱动总裁版本

关于自动构建

我们不可能做到每次 Windows 推出新 Build 时及时更新，我们并没有那么多时间

自定义系统有很多细节，我们不可能做到每一次都记得去处理

当有了新的想法，我们也不能很快的展示给你们

所以自动构建他就来了

有这几点优势，Windows 啥时候更新咱就跟着更新，不用进入系统后再跑一次系Windows Update 了

自动构建后再也不怕忘记处理某处细节了，因我导致的 BUG 也会减少；内置软件的 WTDR.Pack 包也会定时更新；还要咱们的新想法也能够及时的展示给大家了~

啊还有还有，因我导致的 BUG 也会在每次自动构建后进行修复，还会降低 BUG 出现的几率，全自动嘛，就很难出错啦~

Windows 推出新 Build 后，大概24小时内就能构建完成并分发到各个下载节点
