我初次接触到的chrome内核浏览器想起来还是hao123官方的橘子浏览器。

好家伙，它甚至还把我在chrome的收藏偷过来了

当时的官网

当初看到按住快进广告这种功能，这谁顶得住啊，现在想想其实只是一个扩展插件的事。

因为当时年龄问题和国内网络生态我也陆陆续续换过好多家的浏览器，其中不缺360安全浏览器种种

咱先讲一下目前的使用情况吧。

当前使用的浏览器：Chrome

扩展数量：6个

启用扩展：5个

再来讲几个chrome的优点。

  1. 快速, 天下武功无坚不破, 唯快不破. 在同等条件下, Chrome 的加载速度比 IE 能高出一大截. 比 FF 稍微快点.
  2. 兼容. 海纳百川有容乃大. 一个浏览器最核心的素质就在于兼容性. 只要网页内没有 Flash, 无论是你们学校上世纪90年代末上线到现在都没改过一行代码的教案系统, 还是用最近 HTML5 + CSS3 构建的现代网站, Chrome 都能准确无误的显示出来. 相比之下, 同样一个网页, IE8 打开 和 IE11 打开经常是完全不同的效果. 知道十多年前的程序员们是如何针对当时各种版本的 IE 浏览器做兼容性适配的么? 不要问. 他们会哭给你看的.
  3. 安全. 安全是天. 看看 IE 那浑身上下筛子一样的躯体吧. 我记得很清楚，很多的漏洞, 如果你用 IE, 你什么都不做就只是打开带有恶意代码的指定网页, 它就可以通过缓冲区溢出漏洞在你电脑里执行恶意代码. 在QQ授权登陆出现之前, 曾经有网站可以实现 “只要你登录 QQ 且打开指定网站, 它就可以获得你电脑上登录的 QQ 号” 这样神奇的操作. 恐怖如斯.
  4. 丰富. Chrome 是一个伪装成浏览器的瑞士军刀. 据统计, Chrome 扩展应用商店里有大概 188,620 个扩展. 涵盖了生活的方方面面. 自动翻译, 内嵌视频下载, 购物网站比价, 手势控制, 防病毒等等等等… 而且还有油猴这种神器.
  5. 简洁. Simplicity, is hidden complexity. 相比其他浏览器上面好几排按钮, Chrome 就只有三行. 一行 Tab 栏, 一行网址输入框, 一行收藏夹. 让你的注意力集中在内容上. 一个产品做加法不难, 只要把功能都罗列在那里就好了. 界面上密密麻麻的按钮和文字让用户自己去找好了! 真正难的, 是做减法. 该把什么功能全部隐藏起来, 该把什么功能显示出来才是考验一个产品设计师能力的地方.
  6. 克制. 该有的功能它都有, 不该有的功能它一律没有:
    Chrome 不会一天弹八次的 “XX新闻”
    Chrome 不会没事儿在桌面右下角弹出个 “屠龙宝刀, 点击就送”.
    Chrome 不会没事就跳出来跟你讲你的电脑处于危险之中, 赶快安装 XXXXX.
    Chrome 不会从安装到卸载无时无刻想方设法的给你装个 “XX 电脑管家”, “XX 助手”, “XX 蓝月”, “XXX 安全卫士”…
    Chrome 不会在卸载的时候画个裸男假惺惺的哀求你不要卸载它, 更不会把卸载按钮藏得连视力 2.0 的人都难以找到.
  7. 人性. 真正好的产品, 从来都是在你看不见的地方把一切安排的井井有条. 只要你一声召唤, 你需要的东西就会摆在你的面前. 只要用 Google 的账户登录, 你在网站表单上填写的内容, 你的网站访问 Cookie, 你的浏览历史, 就可以无缝同步. 无论在哪台电脑上, 只要打开 Chrome, 都会感觉好像在同一台电脑上使用一样.

何为优秀的产品设计? 此谓也.

我一直认为一款优秀的产品是需要被定义束缚着才能进行可持续发展的.

现在来介绍一下我常用的几款扩展吧.

uBlock Origin

一款高效的网络请求过滤工具，占用极低的内存和 CPU。

我相信各位新装的浏览器第一时间就是安装拦截广告扩展，通常有 ABP 等等

那么为啥要推荐这个扩展，而不是推荐 ABP 呢？

它比 ABP 等拦截软件都要快，都要省资源，相关信息可以看该 Github 项目里的 Wiki

开源，甚至没有捐赠渠道。。。真大佬。。。

功能比 ABP 等扩展多

唯一缺点就是界面不是很好看。。。

不过最近咱们在试用 AdGuard ，好用的话再单独写一篇文章安利给大家~

注意，这个扩展商店里有李鬼扩展，请务必搜索“uBlock Origin”并下载第一个，这个才是真实的

OneTab

节省高达95％的内存，并减轻标签页混乱现象

有时候可能你开了一大堆网页，标签栏已经挤成一堆看不到标题，找不到你想找的标签页，然后这样整理起来就很麻烦了

这时候只要点一下 OneTab ，他就会就会把你打开的标签页整合到一个页面里面，这时候就方便管理了，没用的就点一下 x 将其去掉，要继续使用的就点一下标题重新打开 √

不过大多情况我并不是这样使用的

有时候你打开了一堆网页，专注于某一件事，但是有些突发事件需要关掉浏览器、重启电脑、关机，这时候点一下他就会把所有标签页给收起来，当解决完突发事件需要重新干这件事时，就可以打开 OneTab 恢复之前的标签页组 √

Tampermonkey

The world’s most popular userscript manager

油猴相信大部分人也吃过安利了，这里也不详细介绍了哈，总的来说他可以为网页添加功能、改变样式、屏蔽内容。

这是一部分我在用的油猴脚本，获取渠道大多在 Greasyfork 上

Imagus

Enlarge thumbnails, and show images/videos from links with a mouse hover.

这个！真的特好用，把鼠标悬停在图片上，会显示原尺寸的原图，我就不多演示了，谁用谁知道。

Chrome 扩展商店上有近188,620个扩展，有用/适量的扩展可以帮到你提高生产力、浏览网站体验；过多的扩展反而可能会减慢浏览器运行速度，同时可能会不小心装上恶意扩展而导致个人信息泄露。
