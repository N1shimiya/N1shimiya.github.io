1.0.0 什么是VHD

VHD 是Microsoft Virtual Hard Disk format（微软虚拟磁盘文件）的简称。可以由Microsoft [Virtual PC](https://baike.baidu.com/item/Virtual PC)2007，[Windows Vista](https://baike.baidu.com/item/Windows Vista)，[Windows 7](https://baike.baidu.com/item/Windows 7)/8，Hyper-V，[Windows Server 2008](https://baike.baidu.com/item/Windows Server 2008)R2/2012，Microsoft Virtual Server 2005等创建，[Virtual Box](https://baike.baidu.com/item/Virtual Box)，VMWare等可以挂载使用。

随着虚拟环境的企业工作负荷的增加以及性能要求的提高，虚拟硬盘 (VHD) 格式需要适应这些变化。Windows Server 2012 中的 Hyper-V 引入了一个新版本的 VHD 格式，称为 VHDX，它在设计上可以处理当前以及将来的工作负荷。

与旧的 VHD 格式相比，VHDX 具有更大的存储容量。它还在电源故障期间提供数据损坏保护并且优化动态磁盘和差异磁盘的结构对齐方式，以防止在新的大型扇区物理磁盘上性能降级。

本教程中使用VHDX进行安装 你也可以使用VHD进行安装

1.0.1 VHDX安装系统的特点

  1. 整个系统就是一个文件 方便以后迁移系统
  2. 可以使用VHDX的差分备份 还原速度快
  3. 可以让虚拟机和双系统使用同一VHDX启动 (虽说这次做的不是这种)
